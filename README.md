# House Price Prediction using Machine Learning

## Project Overview
This project builds a Machine Learning model to predict house prices using Multiple Linear Regression. The model analyzes various property and neighborhood features to estimate housing prices accurately.

## Tools and Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Dataset
The dataset contains **2000 housing records** with **15+ features** including:

- Square footage
- Number of bedrooms
- Number of bathrooms
- House age
- Garage spaces
- Distance from city center
- School rating
- Crime rate
- Property tax
- Neighborhood quality

## Project Workflow

1. Data Collection
2. Data Preprocessing
3. Exploratory Data Analysis (EDA)
4. Outlier Detection
5. Feature Scaling
6. Model Training
7. Model Evaluation
8. Cross Validation

## Model Performance

| Metric | Score |
|------|------|
| R² Score | 0.982 |
| MAE | 16100 |
| RMSE | 20641 |
| Cross Validation Score | 0.983 |

## Results
The model explains **98% of the variance in housing prices**, demonstrating strong predictive capability.
